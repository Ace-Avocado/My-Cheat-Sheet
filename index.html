Cheat Sheet

Html BreakDown
DOCTYPE Specifies the version of HTML being used and is placed at the beginning of an HTML document.
HTML Tag The root element of an HTML page. All other elements are contained within theHtmltags.
Head Section Contains metadata about the document, such as the page title, character set, stylesheets, scripts, and other information that isn't directly displayed on the page. It's enclosed within the head tags.
Title Specifies the title of the document, which is displayed on the browser's title bar or tab. It's defined within the titletags inside the headsection.
Body Section Contains the content of the document that is displayed in the browser window. It's enclosed within thebodytags.
CSS BreakDown
Selectors Selectors are used to target HTML elements to apply styles to them. They can be based on element names, classes, IDs, attributes, or even the relationship between elements.
Properties Properties are the styling attributes that you want to apply to the selected elements. Each property has a value associated with it.
Declaration Block A declaration block consists of one or more declarations (property-value pairs) enclosed in curly braces {}. Multiple declarations are separated by semicolons ;.
Rule Set A rule set consists of a selector and a declaration block. It defines the style to be applied to the selected elements.
Inheritence nheritance is the mechanism by which certain properties are passed from parent elements to their children. Not all properties are inherited; only certain ones like font styles, colors, and text alignments.
